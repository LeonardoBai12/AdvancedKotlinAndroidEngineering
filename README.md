# Advanced Kotlin & Android Engineering

*A deep-dive reference covering the Kotlin language, data structures, concurrency, the Android application layer, architecture, testing, and tooling — written for senior-level interview preparation and day-to-day engineering.*

📖 **Read it online:** [leonardobai12.github.io/AdvancedKotlinAndroidEngineering](https://leonardobai12.github.io/AdvancedKotlinAndroidEngineering/)

## About

Hi, I'm **Leonardo Bai**, an Android engineer from Brazil. I built this repository as a structured personal reference — the kind of material I wish had existed when preparing for senior-level technical interviews and deepening my understanding of the Android platform.

The content is compiled from official documentation and a handful of books I consider essential reading for any senior Kotlin/Android engineer:

- [Kotlin Documentation](https://kotlinlang.org/docs/home.html) — JetBrains
- [Android Developers Documentation](https://developer.android.com/docs) — Google
- **Clean Architecture** — Robert C. Martin
- **The Pragmatic Programmer** — David Thomas & Andrew Hunt
